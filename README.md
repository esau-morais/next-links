## 🛠️ Setup

1. If you're already logged in, you'll be able to see the "Use this template" button. Click it and create a new repository

2. Clone

```
gh repo clone esau-morais/next-links
```

3. Installing (using `pnpm`)

```
pnpm install 
```

4. Running (using `pnpm`)

```
pnpm dev
```
